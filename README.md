# IB Academic Planner

## Project Info

A comprehensive academic planning tool for IB curriculum.

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```

## Technologies

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Supabase
